myPlanit

Allows users to create an itinerary of things to do based on the time they have available in a location chosen by them to make the most use out of their time. Users create an account, search for points of interest (poi) and add to their plan. Travel time and directions are generated and subtracted from allocated time given by user. User can determine time to spend at each poi out of remianing time available. Users can save, edit, delete, or post their plans and view others.

  • Angular.js
  • jQuery
  • Underscore
  • Parse
  • Foursquare API
  • Google Maps API
  • Google Directions API
  • Foundation
  • Sass
